## Free Cost Tracker

Use this minimal table when AI coding agents run often enough that token spend, review time, and saved cleanup time need a visible record.

| Field | What To Record |
| --- | --- |
| Date | Run date |
| Agent | Codex, Claude Code, Cursor, or local agent |
| Model | Model or model family |
| Task | Work performed |
| Estimated Tokens | Rough token estimate |
| Direct Cost | Direct dollar cost if known |
| Human Minutes Saved | Estimated time saved |
| ROI Notes | Why the run was or was not worth it |

